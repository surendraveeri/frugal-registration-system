
# Frugal Testing – Registration System Automation

## Overview
This project implements a responsive Registration System with client-side validations
and Selenium automation as part of Frugal Testing Section A assignment.

## Technologies Used
- HTML, CSS, JavaScript
- Java
- Selenium WebDriver
- ChromeDriver

## Features
- Client-side form validation
- Password strength indicator
- Linked Country–State–City dropdowns
- Error handling and success message
- Selenium automation for positive & negative scenarios

## Automation Scenarios
1. Negative test – missing mandatory fields
2. Positive test – successful registration
3. UI validation and form reset verification
